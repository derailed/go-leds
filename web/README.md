# GO-LEDS!

Frontend web app to control and update LED strip colors.

# Assumption

1. NodeJS installed
2. NPM installed

# Running
1. Install npms
```
npm install
```
2. Install 3rd party libs
```
bower install
```
1. Launch the web app
```
gulp
```
2. Visit http://localhost:8000
